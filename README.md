#  Library Borrowing and Reservation System

This project is a web-based system that allows library members to search, borrow, and reserve books, while providing librarians the ability to manage approvals and track book availability.  
The goal is to design a structured and consistent system architecture supported by diagrams, database schemas, and UI/UX designs.

##  Project Description

The Library Borrowing and Reservation System includes:

- Book search by title, author, or category  
- Borrowing and reservation requests  
- Librarian approval workflows  
- Automatic availability updates  
- Borrowing history tracking  
- Due date calculation  
- Validation to prevent double borrowing  
- Secure login and user management  

The project focuses on workflow correctness, validation logic, and data consistency across all system operations.
